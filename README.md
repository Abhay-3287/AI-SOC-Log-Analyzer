# AI-SOC-Log-Analyzer

An AI-powered Security Operations Center (SOC) Log Analyzer that helps security analysts detect suspicious activities by analyzing system and network logs.

## Features

- User Authentication
- Upload SOC Log Files
- AI-Based Log Analysis
- Threat Detection
- Dashboard
- JWT Authentication

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

### AI
- Google Gemini API

## Installation

### Clone Repository

```bash
git clone https://github.com/Abhay-3287/AI-SOC-Log-Analyzer.git
```

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm start
```
